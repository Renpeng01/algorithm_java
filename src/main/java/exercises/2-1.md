#### 2.1.6

插入排序：由于在内层循环条件中有对元素的比较，是n级别

#### 2.1.7
对于逆序数组
选择排序和插入排序的时间复杂度都是n²
选择排序时间复杂度为  (比较) * (1 + n) * n /2 + (交换) * n
插入排序时间复杂度为  (比较 + 交换) * (1 + n) * n /2
综上：选择排序更快
